# Reminder Pro

A modern, sleek, and high-performance reminder application for desktop.

## Features
- **Modern Dark Theme**: A stunning UI designed for focus and clarity.
- **Smart Recurrence**: Daily, weekly, monthly (workdays, weekends), and yearly reminders.
- **Search & Filters**: Easily find and manage your tasks.
- **Stats Dashboard**: Keep track of your progress at a glance.
- **Desktop Notifications**: Built with Electron for seamless desktop integration.

## Getting Started

### Installation
```bash
npm install
```

### Development
```bash
npm start
```
Starts the React dev server.

To run the desktop application:
```bash
npm run electron
```

### Build
To create a production-ready installer:
```bash
npm run dist
```

## Technologies
- **Frontend**: React 19, Tailwind CSS 4, Lucide Icons
- **Desktop Wrapper**: Electron
- **Storage**: LocalStorage
